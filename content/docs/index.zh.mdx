## 概览

Grid Maker 是一款基于浏览器的网格工具，让你上传图片、叠加自定义网格，并导出绘画、素描或排版所需的参考文件。一切在线完成，无需安装，帮你专注比例与构图，而不是手工量格子。

## 核心能力

- **上传与叠加**：JPG/PNG/WebP，最大 15MB，拖拽或文件选择，处理全在本地浏览器。  
- **网格控制**：设置行/列，调整线色、透明度、线宽，开启/关闭标签（A1、B2…）。  
- **纸张规格**：A2/A3/A4/A5、Letter、Legal，或自定义尺寸（毫米）。  
- **预览与滤镜**：实时画布预览，可选黑白转换。  
- **打印指引**：按纸张尺寸自动计算线条数量与间距。  
- **导出**：下载高分辨率网格叠加图片，便于打印或屏幕参考。

## 快速开始

1) 打开 Grid Maker，点击 **Upload Image**（或拖拽文件）。  
2) 设置 **Rows / Columns**，选择 **Line Color、Opacity、Width**，可开启 **Show Labels**。  
3) 选择 **Paper Format**，或自定义宽/高（毫米）。  
4) 预览，按需切换 **黑白**，然后点击 **Download Grid Image**。  
5) 使用 **Printing Guide** 的数值在纸上重现网格，或确认数字绘制的间距。

## 详细设置

- **Grid Type**：自定义或正方形；正方形模式可按行列比例对齐画布。  
- **Labels**：可选单元格标签，适合教学与分步转绘。  
- **Appearance**：通过颜色、透明度、线宽微调，确保参考线不过分干扰画面。  
- **Paper Format**：快速切换常用尺寸；自定义模式支持精确输入毫米尺寸。

## 打印指引（自动）

选择纸张后，Grid Maker 会自动显示：  
- 需要绘制的竖/横线数量与间距。  
- 开启标签时的范围（如 A1 至 D4）。  
可据此在纸上复现网格，或核对数字工作区的间隔。

## 使用提示

- 依据绘图规模选择网格密度：草图用较少格子，细节绘制用更多格子。  
- 打印时可提高线条不透明度；屏上参考可降低不透明度。  
- 正方形模式有助于在不同设备或纸张间保持比例一致。  
- 更改纸张或线条设置后，请重新下载，确保网格匹配最新尺寸。
